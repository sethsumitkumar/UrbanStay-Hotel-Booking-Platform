UrbanStay is a simple hotel booking web application built using HTML, CSS, and JavaScript.  
It allows users to browse hotels, filter by price, and simulate a booking process.

##  Live Demo
https://hotelbookingprojectsumit.netlify.app/

## Features
- View multiple hotel listings
- Filter hotels by price
- Select check-in and check-out dates
- Book a hotel using a modal form
- Booking history saved using LocalStorage
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## About the Project
This project was created to practice frontend development and DOM manipulation.  
It focuses on building a clean UI and implementing basic booking logic using JavaScript.
